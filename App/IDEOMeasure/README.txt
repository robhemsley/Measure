1) Install the following dependencies (Should be in libs Folder already):
	JSON-lib-2.5-jdk15.jar
	Jackson-core-asl.19.7.jar
	Jackson-mapper-asl.1.9.7.jar
	JSON-simple-1.1.1.jar

2) Set the config settings in the res/values/strings.xml file as required
	- Should enter the IP and port that the server is running on
	
3) Build and install the apk
